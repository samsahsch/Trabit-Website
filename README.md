# Trabit — Habit Tracker for iOS

**Track any habit. Reach any goal.**

Trabit is a privacy-first habit tracker built natively for iPhone, iPad, and Apple Watch. It features AI-powered natural language logging, rich analytics, flexible goals, and social accountability — all while keeping your data on your device.

## Website

This repository hosts the [trabit.app](https://trabit.app) website, which serves as the public-facing landing page and handles Universal Links for the iOS app.
